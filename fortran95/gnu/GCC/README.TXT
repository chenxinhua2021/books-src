GCC-2.95 binary release for Mingw distribution. See the INSTALL file for 
more information.

Mingw32 (native) version of GNU compiler toolchain. These tools run
w/out the need of CYGWIN.DLL and also produce code that depends only
on MS-supplied DLLs.

Based on gcc-2.95 for gcc/c++, binutils-2.9.4 (Cygwin b20 source release) 
for gas/ld/libiberty, gdb-4.18 for GDB. See ANNOUNCE for other included 
packages.

See INSTALL so what to download.

Play at your own risk. No Warranties whatsoever.
